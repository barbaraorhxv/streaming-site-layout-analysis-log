# streaming-site-layout-analysis-log
该项目用于分析在线影视网站的页面布局特征，对首页模块排列顺序、分类导航逻辑以及播放页信息展示形式进行归档说明。通过文字归档方式建立长期资料档案，便于研究不同影视平台在结构设计方面的演变。仓库仅保存公开资料信息，不包含视频资源。
